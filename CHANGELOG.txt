Bootloader:
Implemented Power-On Self-Test (POST) error checking
Added support for booting multiple operating systems from a single bootloader
Introduced Rescue Mode, which activates if POST fails, replacing the previous kernel-based maintenance mode
Various bug fixes and stability improvements

Main OS:
Removed maintenance mode due to security concerns and performance issues
Added Virtual RAM Disk to improve overall system efficiency (though it has minimal impact on users directly)

Other Changes:
Added More Applications To Manifest
Improved ReAgent functionality and made it compatible with the following command structure:
Usage: ReAgent [reset, restore2, restore, repair]

General optimizations and refinements
